How to Run Scripts

Running PowerShell scripts is a straightforward process. Here are the steps to follow:

  1. Open PowerShell: To open PowerShell, press the Windows key + R, then type "powershell" and press Enter.
  2. Set Execution Policy: By default, PowerShell only allows you to run signed scripts. To run unsigned scripts, you'll need to set the execution policy to "RemoteSigned" or "Unrestricted". To do this, type "Set-ExecutionPolicy RemoteSigned" or "Set-ExecutionPolicy Unrestricted" and press Enter.
  3. Change to Script Directory: Navigate to the directory where your script is located by typing "cd path\to\directory" and pressing Enter. Alternatively, you can use the "cd" command to change to the directory by typing "cd ." and pressing Tab to autocomplete the directory name.
  4. Run the Script: To run the script, type "./scriptname.ps1" and press Enter. The script will execute and output any results to the PowerShell console.
